# DevNote Functional Specifications

이 문서는 DevNote 애플리케이션의 모든 주요 기능과 요구사항을 정의합니다.

## 1. 개요

DevNote는 개발자가 코드 스니펫과 노트를 효율적으로 관리하고, 다양한 소스(컨플루언스, PDF, Word 등)에서 내용을 쉽게 가져올 수 있도록 돕는 웹 애플리케이션입니다.

## 2. 핵심 기능

### 2.1 노트 관리 (Snippet Management)

- **작성/수정/삭제**: 제목, 마크다운 내용, 코드 스니펫, 카테고리, 태그를 포함한 노트 관리.
- **상세 보기**: 마크다운 렌더링, 코드 하이라이팅, 그리고 **LaTeX 수식 렌더링** 지원.
- **목록 보기**: 상세 모드(내용 포함)와 목록 모드(타이틀 위주) 지원.

### 2.2 폴더 관리 (Category Management)

- **계층 구조**: 폴더 및 하위 폴더 생성 지원.
- **관리 기능**: 이름 변경, 삭제(하위 노트 최상위 이동), 중복 이름 방지.
- **통계**: 폴더별 포함된 노트 개수 실시간 표시.

### 2.3 검색 및 필터링

- **전역 검색**: 제목, 내용, 태그를 통합 검색 (필터 무시).
- **카테고리 필터**: 특정 폴더 및 하위 폴더 데이터만 표시.
- **태그 필터**: 멀티 태깅 지원 및 태그별 필터링.

### 2.4 데이터 연동 (Sync & Import/Export)

- **Cloud Backup**: Firebase Auth 및 Firestore를 이용한 기기간 실시간 자동 동기화.
- **외부 파일 가져오기**: HTML, Word(.docx), PDF 파일에서 텍스트 및 서식 추출 후 마크다운 변환.
- **내보내기**: 전체 노트를 Confluence 호환 HTML 파일로 내보내기.

### 2.5 스마트 에디터 (Rich-Paste)

- **붙여넣기 변환**: 클립보드의 HTML(Gemini, Confluence 등)을 마크다운으로 자동 변환.
- **Confluence 지원**: Confluence Wiki Markup(h1., {code} 등)을 마크다운으로 변환.

## 3. 기술 요구사항

- **PWA**: 오프라인 지원 및 모바일 홈 화면 설치 최적화.
- **버전 관리**:
  - Major.Minor 버전은 별도 요청 전까지 `1.0` 고정.
  - Patch 번호는 `yyyymmddhh24miss` (연월일시분초) 포맷 사용.
  - 전체 형식: `v1.0.yyyymmddhh24miss GMT+9`.
- **디자인**: 미려하고 현대적인 다크/라이트 모드 테마 (Lucide 아이콘 활용).
