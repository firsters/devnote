# DevNote (데브노트) 🚀

개발자를 위한 코드 스니펫 및 메모 관리 웹 애플리케이션입니다. React와 Vite를 기반으로 하며, PWA(Progressive Web App) 기능을 통해 모바일 및 데스크톱에서 네이티브 앱처럼 사용할 수 있습니다.

## ✨ 주요 기능

- **코드 스니펫 관리**: Markdown 문법과 코드 블록을 지원하여 기술적인 메모를 체계적으로 저장합니다.
- **계층형 폴더 시스템**: 드롭다운 형태의 폴더와 하위 폴더 기능을 통해 방대한 노트를 효율적으로 분류합니다.
- **강력한 PWA 통합**: '홈 화면에 추가' 기능을 통해 전체 화면 모드(Standalone)로 실행되며, 오프라인에서도 사용 가능한 사용자 경험을 제공합니다.
- **데이터 내보내기/가져오기**:
  - **Confluence 내보내기**: 저장된 노트를 Confluence와 호환되는 HTML 형식으로 내보낼 수 있습니다.
  - **HTML 가져오기**: 외부 HTML 파일을 읽어와 노트로 변환할 수 있습니다.
- **로컬 스토리지 동기화**: 브라우저 로컬 스토리지를 사용하여 별도의 로그인 없이도 데이터가 유지됩니다.
- **반응형 디자인**: Tailwind CSS를 사용하여 모바일, 태블릿, 데스크톱 등 다양한 기기에 최적화된 UI를 제공합니다.

## 🛠 기술 스택

- **Core**: React 18
- **Build Tool**: Vite
- **Styling**: Tailwind CSS, PostCSS
- **Icons**: Lucide React
- **PWA**: Dynamic Manifest & Meta Injection

## 🚀 시작하기

### 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build
```

### 앱 설치 방법

- **데스크톱**: 브라우저 주소창 우측의 [앱 설치] 아이콘 클릭
- **모바일 (iOS/Android)**: 브라우저 메뉴에서 [홈 화면에 추가] 선택
